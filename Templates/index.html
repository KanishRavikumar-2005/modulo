Build Lightweight Java Backends with Modulo

Modulo is a micro-framework designed for rapid development of lightweight backend logic in pure Java. Ideal for students, prototyping, and small projects, Modulo offers minimal setup and a clean, intuitive API.

Get Started

Key Features

Modulo provides a streamlined approach to backend development with essential features for building efficient and maintainable applications.

Minimal Setup

Get started quickly with Modulo's minimal setup, allowing you to focus on your application logic.

Clean Routing

Define routes cleanly and intuitively, making your application structure easy to understand and manage.

Context-Based Request Handling

Handle requests with context-based processing, simplifying data access and manipulation.

Session Management

Manage user sessions without external dependencies, keeping your application lightweight and self-contained.

Additional Features

Modulo includes a range of additional features to enhance your development experience and application capabilities.

Dynamic Routes

Create dynamic routes that adapt to different parameters, providing flexibility in your application's URL structure.

Static File Serving

Serve static files directly from your application, simplifying the management of assets like HTML, CSS, and JavaScript.

Cookies

Manage cookies easily, allowing you to store and retrieve data on the client-side.

Form/Query Parsing

Parse form data and query parameters effortlessly, streamlining the handling of user input.

Getting Started

Follow these steps to start building your first Modulo application:

1. Installation

Install the Modulo CLI using your preferred package manager.

2. Git Clone

Clone the Modulo repository using the command: git clone [repository URL]

3. Create Routes

Create new routes by defining handlers for different URL paths in your application.

4. Create HTML Views

Set up basic HTML views by creating HTML files and linking them to your routes for rendering.

5. Development

Start the development server and begin coding your application.
